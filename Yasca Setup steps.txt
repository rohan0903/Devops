1.Download the yasca-core-2.21 zip folder
2.unzip the folder 
3.Install the php 5 in the folder 
Installation steps:-
wget http://in1.php.net/distributions/php-5.3.29.tar.bz2
tar -xvf php-5.3.29.tar.bz2
apt-get update
apt-get install build-essential
apt-get install libxml2-dev
cd php-5.3.29
./configure
make
make test # Neither mandatory nor a bad idea.
sudo make install
/usr/local/bin/php -v
4.Create a yascaConfigScript.sh file
5.Add these lines in the above file
